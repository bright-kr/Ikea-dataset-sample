# # Ikea-dataset-samples

<h2>1001개 Ikea 제품으로 구성된 샘플 データセット</h2>

![Ikea dataset header](https://github.com/luminati-io/Ikea-dataset-sample/blob/main/Ikea-datasets.png)

1000개 이상의 레코드로 구성된 Ikea データセット 샘플입니다. データセット은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>Ikea データセット에 포함된 일부 데이터 포인트는 다음과 같습니다:</h2>

* ```description```: Ikea 제품을 설명하는 텍스트 상세 정보입니다
* ```in_stock```: 제품이 현재 재고가 있는지 여부를 나타냅니다
* ```color```: 제품에서 사용 가능한 색상 또는 색상 옵션입니다
* ```size```: 제품의 크기 또는 크기 옵션입니다
* ```reviews_count```: 제품의 총 리뷰 수입니다
* ```main_image```: 제품의 대표 이미지에 대한 참조 또는 링크입니다
* ```category_url```: 제품 카테고리로 연결되는 URL 링크입니다
* ```category```: 제품이 속한 카테고리입니다
* ```category_id```: 카테고리의 고유 식별자입니다
* ```category_tree```: 제품 카테고리를 계층적으로 표현한 정보입니다
* ```rating```: 제품의 평균 평점입니다
* ```image_urls```: 제품 이미지에 대한 참조 또는 링크입니다
* ```designer```: 제품과 관련된 디자이너입니다
* ```details```: 제품에 대한 추가 상세 정보입니다
* ```brand```: 제품과 관련된 브랜드입니다
* ```related_products```: 현재 제품과 관련된 제품입니다
* ```model_number```: 제품의 모델 번호입니다
* ```video_count```: 제품과 관련된 총 비디오 수입니다
* ```videos```: 제품과 관련된 비디오에 대한 참조 또는 링크입니다
* ```image_count```: 제품과 관련된 총 이미지 수입니다
* ```badges```: 제품과 관련된 배지 또는 라벨입니다
* ```care```: 제품의 관리 지침 또는 관련 상세 정보입니다
* ```documents```: 제품과 관련된 문서입니다
* ```facts```: 제품에 대한 사실 또는 특정 정보입니다
* ```materials_description```: 제품에 사용된 소재에 대한 설명입니다
* ```measurements```: 제품의 치수 또는 측정값입니다
* ```model_name```: 제품 모델의 이름 또는 제목입니다
* ```offers```: 제품과 관련된 오퍼 또는 프로모션입니다
* ```other_attributes```: 제품의 추가 속성 또는 기능입니다
* ```packages_description```: 제품 포장에 대한 설명입니다
* ```product_series```: 제품이 속한 시리즈입니다

그리고 더 많은 항목이 있습니다.

이는  <b>71.9K 레코드</b> 이상을 포함하는 "Ikea products"
データセット에서 파생된 샘플 하위 집합입니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. 옵션으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>요청에 따라 제공됩니다.</b>

<b>[전체 Ikea データセット 받기](https://brightdata.co.kr/products/datasets/ikea)</b>.

<h2>Ikea データセット의 사용 사례는 무엇입니까?</h2>

<h3>1. Pricing Optimization</h3>
유사한 IKEA 제품 및 카테고리를 경쟁사의 제품 및 카테고리와 비교하여 가격 전략을 수립하고 동적 가격 책정 모델을 구현합니다.

<h3>2. Identify Gaps</h3>
재고 격차, 특정 IKEA 제품에 대한 수요 증가, 소비자 사이에서의 트렌딩 아이템을 식별합니다.

<h3>3. Market Strategy Optimization</h3>
IKEA データセット을 활용하여 시장 전략을 분석하고, 핵심 트렌드를 발굴하며, 고객 선호도를 이해합니다.
